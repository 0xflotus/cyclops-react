## Maven

```xml

    <groupId>com.aol.cyclops</groupId>
    <artifactId>cyclops-guava</artifactId>
    <version>x.y.z</version>

```

# Features

Currently requires Guava 18.0 or above

Use Guava.anyM to create wrapped Guava Monads.

* Javaslang
